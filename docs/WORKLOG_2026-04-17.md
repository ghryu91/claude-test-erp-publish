## 9. 변수 네이밍 규칙: 축약 지양

팀 합의 기반 규칙 적용 — 사용자가 축약형을 선호하지 않음 명시.

### 적용 규칙
- `res` → `response`, `err` → `error`, `ctx` → `context`, `req` → `request`
- 넓은 스코프 반복문 인덱스 `i`/`pgm` 등도 의미 있는 이름으로 (`index`, `program`)
- shadcn/ui 원본 컴포넌트(`copy-button.tsx`)는 라이브러리 업데이트 충돌 방지를 위해 제외
- React 관용 표현(`setState((prev) => ...)`)의 `prev`는 보존

### 수정 파일
- `src/pages/user/UserPage.tsx` (`masterRes`→`masterResponse`, `affRes`→`affiliatesResponse`, `pgm`→`program`)
- `src/pages/customer/CustomerRegisterPage.tsx`
- `src/pages/system/RemoteDbQueryPage.tsx`
- `src/pages/user/UserSearchPage.tsx` (`axiosErr`→`axiosError`, 2곳)
- `src/pages/auth/SignInPage.tsx`
- `src/contexts/AuthContext.tsx` (`req`/`res`/`ctx` 전부)
- `src/api/authService.ts`
